# 🍕 Pizza Sales Analysis Project

## 📊 Project Overview

This project analyzes pizza sales data using **SQL Server** and **Power BI** to extract meaningful business insights.
The goal is to understand sales performance, customer behavior, and product trends to support data-driven decisions.

---

## 🛠️ Tools & Technologies

* SQL Server – Data querying and analysis
* Power BI – Data visualization & dashboard creation
* Excel/CSV Dataset – Source data

---

## 📁 Project Files

* `Pizza Sales Analysis Project.sql` → SQL queries for analysis
* `Pizza Sales Analysis Project.pbix` → Power BI dashboard
* `PIZZA SALES SQL QUERIES.docx` → Query documentation

---

## 📌 Key Performance Indicators (KPIs)

* Total Revenue
* Average Order Value
* Total Pizzas Sold
* Total Orders
* Average Pizzas Per Order

---

## 📈 Analysis Performed

### 1. Sales Trends

* Daily order trends
* Monthly order trends

### 2. Revenue Insights

* % of sales by pizza category
* % of sales by pizza size

### 3. Product Performance

* Top 5 pizzas by revenue
* Bottom 5 pizzas by revenue
* Top & bottom pizzas by quantity
* Top & bottom pizzas by total orders

---

## 🧠 Key SQL Concepts Used

* Aggregate Functions (SUM, COUNT, AVG)
* GROUP BY and ORDER BY
* Subqueries
* Type Casting (CAST)
* Date Functions (DATENAME, MONTH)

---

## 📊 Dashboard Features (Power BI)

* Interactive filters (category, size, date)
* KPI cards for quick insights
* Trend analysis charts
* Top & bottom product visuals

---

## 🎯 Business Insights

* Identify best-selling pizzas
* Understand customer ordering patterns
* Analyze peak sales days/months
* Optimize menu and pricing strategy

---

## 🚀 How to Use

1. Import dataset into SQL Server
2. Run SQL queries from `.sql` file
3. Load processed data into Power BI
4. Open `.pbix` file to explore dashboard

---

## 📌 Conclusion

This project demonstrates how raw data can be transformed into actionable insights using SQL and Power BI. It is useful for improving sales strategies and business performance.
